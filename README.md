# BANK SAMPAH GT

Aplikasi pengelolaan bank sampah menggunakan Flutter.

Fitur :

- Tambah / Edit / Hapus Jenis Sampah.
- Tambah / Edit / Hapus Harga Jenis Sampah.
- Daftar Anggota.
- Transaksi Bank Sampah.
